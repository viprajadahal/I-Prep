# I-Prep
I-Prep is a full-stack web application for IELTS Preparation and Performance Analysis using Natural Language Processing (NLP)

## Project Overview
This system is designed to help IELTS aspirants prepare effectively by providing structured practice across all four exam modules Reading, Writing, Listening, and Speaking along with NLP-driven feedback and performance analytics that track progress over time and identify areas needing improvement.

Built as a 5th Semester Project at **Kathmandu University, Department of Computer Science and Engineering.**
